# Voya AI - Lodge Management Platform

This is a premier, safari-themed management platform for lodge managers.

## Project Structure
- **/frontend**: Next.js 14 dashboard using Tailwind CSS and Clerk.
- **/backend**: Fastify REST API with TypeScript, Supabase, and M-Pesa Daraja integration.

## Quick Start
### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Deployment
- **Frontend**: Deployed on **Vercel**.
- **Backend**: Configured for **Render.com** (see `backend/render.yaml`).

## Branding
The project uses a custom Silicon Valley style logo (found in `frontend/public/logo.png`) and a premium earthy aesthetic.
